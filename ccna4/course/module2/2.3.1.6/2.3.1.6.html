Verificación de la configuración de PPP

Utilice el comando show interfaces serial para verificar la configuración de la encapsulación PPP o HDLC. El resultado del comando en la figura 1 muestra una configuración PPP.

Cuando configure HDLC, la salida del comando show interfaces serial debe mostrar encapsulation HDLC. Cuando se configura PPP, también se muestran los estados de LCP y NCP. Observe que los protocolos NCP IPCP e IPV6CP están abiertos para IPv4 e IPv6, ya que el R1 y el R2 se configuraron con direcciones IPv4 e IPv6.

En la figura 2, se resumen los comandos que se usan para verificar PPP.

El comando show ppp multilink verifica que el protocolo PPP multienlace esté habilitado en el R3, como se muestra en la figura 3. El resultado indica la interfaz Multilink 1, los nombres de host de las terminales locales y remotas, y las interfaces seriales asignadas al grupo multienlace.

La figura 1 muestra la salida del comando show interfaces serial cero barra diagonal cero barra diagonal cero. La figura resalta la línea que indica que la interfaz está activa. Esta sección de la salida dice encapsulation ppp, lcp, open, y las listas abiertas para IP versión 4 e IP versión 6. La figura 2 muestra los tres comandos que se usan para verificar una conexión PPP. Esta lista incluye el comando show interfaces para ver todas las interfaces en un router, el comando show interfaces serial para dar información sobre una interfaz serial y el comando show ppp multilink para mostrar información sobre una interfaz PPP multienlace. La figura 3 muestra la salida del comando show ppp multilink. Esta salida muestra información sobre el multienlace. Esta salida también incluye nombres de host de los terminales locales y remotos, y los enlaces seriales que se encuentran activos en el paquete multienlace.
