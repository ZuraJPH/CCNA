La figura 1 muestra el inicio de una conexión PPP.  En la figura, el router 1 intenta conectarse al router 3.  El router 1 envía un nombre de usuario y una contraseña al router 3.  La figura 2 muestra que el router 3 evaluará el nombre de usuario y la contraseña mediante su base de datos local. El router 3 acepta o rechaza la conexión.  Este es el intercambio bidireccional de PAP.
