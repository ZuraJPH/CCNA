La figura muestra los siguientes parámetros para el comando debug PPP: paquete, negociación, error, autenticación, compresión y CDCP.  El paquete se utiliza para mostrar los paquetes de PPP que se envían y reciben. La negociación se utiliza para mostrar los paquetes de PPP durante el inicio de PPP. El error se utiliza para mostrar errores de protocolo. La autenticación se utiliza para mostrar mensajes de protocolo de autenticación incluidos los intercambios de CHAP y de PAP. La compresión se utiliza para mostrar información específica al intercambio de las conexiones PPP mediante MPPC. CDCP se utiliza para mostrar los errores y las estadísticas del protocolo asociados con las negociaciones de conexiones PPP mediante MSCB.
