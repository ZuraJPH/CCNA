Las figuras 1 a 3 muestran el inicio de una conexión CHAP entre dos routers.  En la figura 1, el router 3 inicia el intercambio de tres vías y envía un mensaje de desafío al router 1. La figura 2 muestra el router 1 que responde a la solicitud de desafío enviando su nombre de usuario de CHAP junto con un valor hash para la contraseña. La figura 3 muestra el router 3 que evalúa el nombre de usuario y valida el hash comparándolo con el hash de la contraseña que se encuentra en la base de datos.
