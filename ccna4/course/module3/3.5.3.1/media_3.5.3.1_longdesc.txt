Una tabla de dos columnas muestra los comandos de configuración de BGP.  Las dos columnas son Comando y Descripción. La primera fila es el comando router bgp as-number que permite un proceso de routing BGP, y coloca al router en modo de configuración de router. El segundo comando es neighbor ip-address remote-as as-number que se utiliza para especificar un vecino de BGP. El as-number es el número de AS del router vecino. La última fila contiene la dirección de red dirección-de-red y, opcionalmente, el comando mask máscara-de-red que se usa para anunciar una dirección de red a un vecino de eBGP como originada por este AS. La máscara-de-red es la máscara de subred de la red.
