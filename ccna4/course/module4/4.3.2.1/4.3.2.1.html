Configuración de topología de IPv6

En la figura 1, se muestra la topología que se utilizará para configurar las ACL de IPv6. Esta es similar a la topología de IPv4 anterior, excepto por el esquema de direccionamiento IPv6. Existen tres subredes 2001:DB8:CAFE::/64:

  • 2001:DB8:CAFE:10::/64

  • 2001:DB8:CAFE:11::/64

  • 2001:DB8:CAFE:30::/64

Dos redes seriales conectan los tres routers:

  • 2001:DB8:FEED:1::/64

  • 2001:DB8:FEED:2::/64

En las figuras 2, 3 y 4, se muestra la configuración de la dirección IPv6 para cada uno de los routers. El comando show ipv6 interface brief se utiliza para verificar la dirección y el estado de la interfaz.

Nota: el comando no shutdown y el comando clock rate no se muestran.

La figura 1 muestra la red de referencia de ACL de tres routers. R2 se encuentra en la parte superior de un triángulo y tiene una conexión WAN serie tanto con R1 como con R3. No hay conexión directa entre R1 y R3. R1 tiene dos interfaces gigabit activas para los segmentos de red locales y cada segmento tiene un switch con una computadora conectada a él. R-1 tiene la red ipv6 2 0 0 1 dos puntos D B 8 dos puntos C A F E dos puntos 1 0 dos puntos dos puntos barra diagonal 64 en la interfaz gigabit cero barra diagonal cero, y la red ipv6 2 0 0 1 dos puntos D B 8 dos puntos C A F E dos puntos 1 1 dos puntos dos puntos barra diagonal 64 en la interfaz gigabit cero barra diagonal uno. R-3 tiene una sola interfaz activa con la red ipv6 2 0 0 1 dos puntos D B 8 dos puntos C A F E dos puntos 3 0 dos puntos dos puntos barra diagonal 64 en la interfaz gigabit cero barra diagonal uno. La WAN serial entre R-1 y R-2 es 2 0 0 1 dos puntos D B 8 dos puntos F E E D dos puntos 1 dos puntos dos puntos barra diagonal 64. La WAN serial entre R-2 y R-3 es 2 0 0 1 dos puntos D B 8 dos puntos F E E D dos puntos 2 dos puntos dos puntos barra diagonal 64. La figura 2 muestra los comandos que se usan para configurar cada interfaz en R1. El primer comando es: interface gig cero barra diagonal cero. El siguiente comando es ipv6 address 2 0 0 1 dos puntos D B 8 dos puntos C A F E dos puntos 1 0 dos puntos dos puntos uno barra diagonal 64. La interfaz siguiente es 0 barra diagonal cero barra diagonal cero y el comando para la dirección es ipv6 address 2 0 0 1 dos puntos D B 8 dos puntos F E E D dos puntos 1 dos puntos dos puntos uno barra diagonal 64. La interfaz siguiente es la interfaz gigabit Ethernet cero barra diagonal uno y el comando de asignación de direcciones es ipv6 address 2 0 0 1 dos puntos D B 8 dos puntos C A F E dos puntos 1 1 dos puntos dos puntos uno barra diagonal 64. Los comandos se verifican mediante el comando show ipv6 interface brief. Esta salida muestra las interfaces IPv6 activas y sus direcciones.
