La figura 1 muestra la red de referencia de ACL de tres routers.  R2 se encuentra en la parte superior de un triángulo y tiene una conexión WAN serie tanto con R1 como con R3.  No hay conexión directa entre R1 y R3.  R1 tiene dos interfaces gigabit activas para los segmentos de red locales y cada segmento tiene un switch con una computadora conectada a él.  R-1 tiene la red ipv6 2 0 0 1 dos puntos D B 8 dos puntos C A F E dos puntos 1 0 dos puntos dos puntos barra diagonal 64 en la interfaz gigabit cero barra diagonal cero, y la red ipv6 2 0 0 1 dos puntos D B 8 dos puntos C A F E dos puntos 1 1 dos puntos dos puntos barra diagonal 64 en la interfaz gigabit cero barra diagonal uno.  R-3 tiene una sola interfaz activa con la red ipv6 2 0 0 1 dos puntos D B 8 dos puntos C A F E dos puntos 3 0 dos puntos dos puntos barra diagonal 64 en la interfaz gigabit cero barra diagonal uno.  La WAN serial entre R-1 y R-2 es 2 0 0 1 dos puntos D B 8 dos puntos F E E D dos puntos 1 dos puntos dos puntos barra diagonal 64. La WAN serial entre R-2 y R-3 es 2 0 0 1 dos puntos D B 8 dos puntos F E E D dos puntos 2 dos puntos dos puntos barra diagonal 64.
