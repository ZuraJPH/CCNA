La figura 1 muestra la sintaxis de los comandos comunes y opciones para una ACL IPv6.  El comando comienza con la frase clave: ipv6 access guion list.  A esto le sigue el nombre de la ACL.  Luego, una instrucción de permiso o denegación, el protocolo, las direcciones de origen y de destino, la palabra clave any para que coincidan todas las direcciones, o la palabra clave host para aplicarla a un host específico, un operador como eq y el nombre o el número del puerto opcional. La figura 2 muestra un ejemplo que crea una ACL IPv6.  El primer comando es: ipv6 access guion list no guion R3 guion lan guion access.  El siguiente comando es: deny ipv6 2 0 0 1 dos puntos d b 8 dos puntos c a f e dos puntos 3 0 dos puntos dos puntos barra diagonal 30 any.  El último comando es: permit ipv6 any any.
