Las imágenes de esta página proporcionan un resumen de los tres modelos diferentes de red. En la Figura 1 podemos ver varios clientes y servidores conectados al mismo switch. Con el cliente y el servidor de archivos, el servidor de archivos almacena el archivo. Los clientes acceden al archivo mediante un software de cliente, como Windows Explorer. Con el cliente y el servidor web, el servidor web ejecuta el software de servidor. Los clientes usan un software de navegador, como Windows Internet Explorer, para acceder a páginas web en el servidor. Con el cliente y el servidor de correo electrónico, el servidor de correo electrónico ejecuta el software de servidor. Los clientes usan un software de correo electrónico, como Microsoft Outlook, para acceder al correo electrónico en el servidor. La figura 2 muestra los siguientes dispositivos y servicios conectados a la nube: móvil, computadora portátil, servidor, red, PC y base de datos. La figura 3 muestra la computación en la niebla para un aeropuerto, edificios de departamentos y un restaurante, y la manera en que todos se conectan a Internet para obtener servicios de computación en la nube y de centro de datos.
