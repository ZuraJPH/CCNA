La figura de instalación del SO de hipervisor muestra la estructura del hipervisor en dos servidores. El componente de hardware se encuentra en el nivel inferior, con el hipervisor instalado sobre el hardware. Cuatro instancias del sistema operativo están conectadas con el hipervisor. Para el servidor 1, hay cuatro instancias de SO Windows. En el servidor 1, sobre las cuatro instancias de SO Windows, están los servicios para servidor web, servidor de correo electrónico, servidor de SQL y servidor de archivos. Para el servidor 2, hay dos instancias de SO Windows y dos instancias de un SO Linux. En el servidor 2, sobre las dos instancias de SO Windows, están los servicios para el servidor de DHCP y el servidor de AD. En el servidor 2, sobre las dos instancias de SO Linux, están los servicios de servidor Radius y servidor NMS.
