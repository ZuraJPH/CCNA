En la figura uno, se muestra la topología de referencia de OSPF. Los tres routers están configurados en un diseño triangular. Cada router está conectado con los otros dos routers mediante enlaces WAN seriales.  El enlace entre el router R1 y el router R2 tiene una velocidad de 1544 kilo bps; el enlace entre el R2 y el R3 funciona a una velocidad de 1024 kilo bps; y el enlace entre el R1 y el R3 funciona a una velocidad de 64 kilo bps.  Cada router tiene una LAN local configurada en cada interfaz gigabit Ethernet cero barra diagonal cero.  Uno de los tres routers tiene una interfaz de loopback configurada que simula una conexión a Internet.  Los tres routers están en el área 0 de OSPF. En la figura dos, se muestra la salida del comando show ip route barra vertical include 172 punto 16 punto 2 punto cero.  En la figura se destaca la métrica para llegar a la red y el valor es 65.  En la figura, luego se muestra la salida del comando show ip route 172 punto 16 punto 2 punto cero. En la salida, se resalta la sección que indica “route metric is 65” (la métrica de ruta es 65).
