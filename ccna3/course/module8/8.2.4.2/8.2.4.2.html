Verificación de la configuración del protocolo OSPF

Como se muestra en la figura 1, el comando show ip protocols es una manera rápida de verificar la información fundamental de configuración OSPF. Esto incluye la ID del proceso OSPFv2, la ID del router, las redes que anuncia el router, los vecinos de los que el router recibe actualizaciones y la distancia administrativa predeterminada, que para OSPF es de 110.

Utilice el verificador de sintaxis de la figura 2 para verificar la configuración del protocolo OSPFv2 del R2 y el R3 con el comando show ip protocols.

En la figura uno, se muestra la salida del siguiente comando ejecutado en el router R1: show ip protocols. La salida muestra la ID del proceso OSPF, la ID del router, las redes que anuncia el router, los vecinos de los que el router recibe actualizaciones y la distancia administrativa predeterminada. La figura dos es un elemento de medios interactivos que permite que el estudiante verifique la configuración del protocolo OSPF.
