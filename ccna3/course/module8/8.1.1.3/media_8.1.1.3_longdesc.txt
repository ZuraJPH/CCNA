La figura uno presenta una tabla que enumera las estructuras de datos de OSPF.  OSPF crea una base de datos de adyacencias, la que contiene la tabla de vecinos y enumera a todos los routers vecinos.  El comando show ip ospf neighbor se utiliza para ver la tabla.  OSPF crea una base de datos de estado de enlace que contiene la tabla de topología e indica información acerca de todos los demás routers en la red.  El comando show ip ospf database se utiliza para ver esta tabla.  OSPF también crea una base de datos de reenvío que contiene la tabla de routing.  En ella se enumeran las rutas generadas cuando se ejecuta un algoritmo en la base de datos de estado de enlace.  El comando show ip route se utiliza para ver esta tabla. En la figura dos, se enumeran los paquetes que intercambian los routers OSPF.
