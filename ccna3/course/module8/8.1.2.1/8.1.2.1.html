Encapsulación de mensajes OSPF

Los mensajes OSPFv2 que se transmiten a través de un enlace Ethernet contienen la siguiente información:

  • Encabezado de la trama de Ethernet de enlace de datos: identifica las direcciones MAC de multidifusión de destino 01-00-5E-00-00-05 o 01-00-5E-00-00-06 al encapsular un mensaje OSPFv2. (figura 1)

  • Encabezado de paquetes IPv4: identifica la dirección IP de origen y la dirección de destino. La dirección de destino es una de dos direcciones de multidifusión OSPFv2: 224.0.0.5 o 224.0.0.6. El encabezado también contiene un campo de protocolo que incluirá el código "89" para OSPF. (figura 2)

  • Encabezado del paquete OSPF: identifica el tipo de paquete OSPF, la ID del router y la ID del área. (figura 3)

  • Datos específicos del tipo de paquete OSPF: contiene información del tipo de paquete OSPF. El contenido varía según el tipo de paquete. (figura 4)

En la figura uno, se muestra el encabezado de la trama de Ethernet de enlace de datos y se resaltan la dirección MAC de multidifusión de destino y la dirección MAC de origen. En la figura dos, se resalta el encabezado de paquetes IP, incluidas las direcciones IP de origen y destino. En la figura tres, se resalta el encabezado de paquetes OSPF que permite identificar el tipo de paquete OSPF, la ID del router y la ID del área. En la figura cuatro, se resalta la información del tipo de paquete OSPF.
