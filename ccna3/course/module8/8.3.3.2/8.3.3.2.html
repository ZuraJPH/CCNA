Verificación de la configuración del protocolo OSPFv3

Como se muestra en la figura 1, el comando show ipv6 protocols es una manera rápida de verificar la información fundamental de configuración de OSPFv3, incluidas la ID del proceso OSPFv3, la ID del router y las interfaces habilitadas para OSPFv3.

Utilice el verificador de sintaxis de la figura 2 para verificar la configuración del protocolo OSPFv3 del R2 y el R3 con el comando show ipv6 protocols.

Además, utilice el comando show ipv6 ospf para examinar la ID del proceso OSPFv3 y la ID del router. Este comando muestra información de área OSPFv3 y la última vez que se calculó el algoritmo SPF.

En la figura uno, se muestra la salida del siguiente comando ejecutado en el router R1: show ipv6 protocols. La salida muestra la siguiente información fundamental de configuración de OSPFv3: la ID del proceso de OSPF, la ID del router y las interfaces habilitadas para OSPFv3. La figura dos es un elemento de medios interactivos que permite que el estudiante verifique la configuración de protocolo OSPFv3 para el router R2.
