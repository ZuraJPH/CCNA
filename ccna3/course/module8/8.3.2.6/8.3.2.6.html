Habilitación de OSPFv3 en las interfaces

OSPFv3 utiliza un método diferente para habilitar una interfaz para OSPF. En lugar de usar el comando network del modo de configuración del router para especificar las direcciones de interfaz que coinciden, OSPFv3 se configura directamente en la interfaz.

Para habilitar OSPFv3 en una interfaz, utilice el comando ipv6 ospf process-id area area-id del modo de configuración de interfaces.

El valor process-id identifica el proceso de routing específico y debe coincidir con la ID de proceso utilizada para crear el proceso de routing en el comando ipv6 router ospf process-id .

El valor area-id es el área que se debe asociar a la interfaz OSPFv3. Aunque pudo haberse configurado cualquier valor para el área, se seleccionó 0 debido a que el área 0 es el área troncal a la que se deben conectar todas las demás áreas, como se muestra en la figura 1. Esto contribuye a la migración a OSPF multiárea, si surge la necesidad.

En la figura 2, se habilita OSPFv3 en las interfaces del R1 con el comando ipv6 ospf 10 area 0. El comando show ipv6 ospf interface brief muestra las interfaces OSPFv3 activas.

Utilice el verificador de sintaxis de la figura 3 para habilitar OSPFv3 en las interfaces del R2 y el R3.

En la figura uno, se muestra la topología de referencia de OSPF. En la figura dos, se muestran los comandos necesarios para habilitar OSPFv3 en las interfaces del router R1. La figura comienza en el modo de configuración global. El primer comando es interface gigabit ethernet cero barra diagonal cero. Esto es seguido por el comando ipv6 ospf 10 área cero. La figura luego pasa a la interfaz serial cero barra diagonal cero barra diagonal cero y repite el comando ipv6 ospf 10 área cero. La figura luego pasa a la interfaz serial cero barra diagonal cero barra diagonal uno y repite el comando ipv6 ospf 10 área cero. Se muestra la salida del comando show ipv6 ospf interfaces brief. La salida muestra que las tres interfaces están en el área cero y participan en OSPFv3. La figura tres es un elemento de medios interactivos que permite que el estudiante habilite OSPFv3 en las interfaces del router R2.
