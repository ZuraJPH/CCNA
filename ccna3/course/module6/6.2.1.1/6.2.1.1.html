Topología de la red EIGRP

En la figura 1, se muestra la topología que se usa en este capítulo para configurar EIGRP para IPv4.

Los routers en la topología tienen una configuración inicial que incluye las direcciones de las interfaces. En este momento, ninguno de los routers tiene configurado routing estático o routing dinámico.

En las figuras 2, 3 y 4, se muestran las configuraciones de las interfaces para los tres routers EIGRP en la topología. Solo los routers R1, R2, y R3 forman parte del dominio de routing EIGRP. El router ISP se usa como gateway del dominio de routing a Internet.

Estas figuras se basan en el diagrama de laboratorio estándar con un enlace adicional entre el router uno y el router tres. En la figura uno, se muestra el diagrama de laboratorio con una conexión del router dos a Internet a través de ISP. Todos los enlaces seriales se etiquetan con el ancho de banda, la identificación de red y el prefijo. En la figura dos, se muestra la salida que genera el comando “show running dash config” en el router uno. Aquí se incluyen las interfaces y su configuración IP actual. En la figura tres, se muestra la salida que genera el comando "show running dash config" en el router dos. Aquí se incluyen las interfaces y su configuración IP actual. En la figura cuatro, se muestra la salida que genera el comando “show running dash config” en el router tres. Aquí se incluyen las interfaces y su configuración IP actual.
