La figura 1 muestra tres switches interconectados mediante enlaces troncales.  El switch uno tiene una referencia que muestra el uso del comando spanning guion tree vlan one root primary.  Esto garantiza que el switch uno se convierta en el puente raíz.  El switch tres tiene una referencia que muestra el uso del comando spanning guion tree vlan one priority.  Esto le permite establecer la prioridad del puente para las elecciones del puente raíz.  El switch dos tiene una referencia que muestra el uso del comando spanning guion tree vlan one root secondary.  Esto permitirá que el switch dos se convierta en el puente raíz en caso de falla del puente raíz actual. La figura 2 muestra el resultado del comando show spanning guion tree. La figura 3 es una actividad interactiva que permite que el estudiante practique la configuración de la prioridad de puente.
